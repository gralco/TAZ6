About Sample Prints

This directory contains freely licensed sample prints for your LulzBot TAZ 6 desktop 3D printer. Each one comes with four files:

  • Attribution: Credit to the original creator of the model.
  • License: The copyleft terms that protect this model file and ensure it can be freely used.
  • STL: Model files that can be edited in your 3D modeling software of choice and/or uploaded into Cura LulzBot Edition.
  • GCODE: Printing files prepared with Cura LulzBot Edition quickprint profiles, based on the STL.

All GCODE in this directory is generated from nGen (colorFabb) quickprint profiles in Cura LulzBot Edition. To learn more about the software used to run your LulzBot TAZ 6, visit LulzBot.com/Cura

nGen is the default material for the LulzBot TAZ 6. nGen is a co-polyester filament made from Eastman Amphora 3300. To learn more about nGen and other materials, visit LulzBot.com/filament
